# Приложение для ребенка

### decsription
# помощь ребенку в выполнении ежедневных действий:
# - чистка зубов
# - чтение
# - зарядка
# - еда
# - прибирание игрушек

# каждое действие имеет таймер выполения
# после выполнения действия - начисляются монетки

### Stack
#### Android Studio, Flutter, Dart, Github
